## Content Is Not the Only Thing

Using Codex, writing code has never felt this easy.

For a long time, I thought the content of a blog post was the only thing that really mattered. The design of the website felt secondary, maybe even a distraction.

I do not think that anymore.

With Codex, I can keep improving my own site whenever I want. Changing the design, layout, and details is now so easy that I actually want to keep refining it. I will definitely continue customizing my website and making it look better.

## Beautiful by Default

The bigger idea here is "beautiful by default."

Code agents are raising the baseline for software development. When it becomes easy to improve how software looks and feels, we should become more selective. We should expect better software, not just software that works.

## Appendix

I use AI to help me write because AI can write better than I can.

That is partly true, but not in the way people usually mean it. What I really provide is the idea. AI helps turn that idea into clearer, easier-to-understand sentences.

I think that is a good way to use it.

## 内容并不是唯一重要的事

用了 Codex 之后，写代码这件事从来没有像现在这么轻松过。

很长一段时间里，我都觉得一篇博客最重要的只有内容本身。至于网站的设计，我一直觉得那只是次要的，甚至有点像干扰项。

但我现在不这么想了。

有了 Codex，我可以随时继续改进自己的网站。无论是设计、布局，还是各种细节，现在改起来都非常容易，以至于我真的会想持续打磨它。我肯定会继续定制我的网站，让它变得更好看。

## 默认就该是美的

这里更大的想法是：“beautiful by default”。

代码代理正在把软件开发的基线抬高。当改进软件的外观和体验变得容易之后，我们就应该变得更挑剔一些。我们应该期待更好的软件，而不只是“能用”的软件。

## 附录

我会用 AI 来帮助我写作，因为 AI 确实比我更会组织句子。

这句话有一部分是对的，但并不是通常理解的那种意思。真正由我提供的是想法，而 AI 帮助我把这些想法整理成更清晰、更容易理解的句子。

我觉得这是一种很好的使用方式。
